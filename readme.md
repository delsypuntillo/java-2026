## 📚 Descripción del Curso

El curso de **Desarrollo de Aplicaciones** tiene como finalidad aprender los fundamentos de programación, diseño y construcción de aplicaciones utilizando el lenguaje de programación **Java**.

Durante el desarrollo del curso se aplican conceptos como:

* Programación orientada a objetos (POO)
* Uso de clases y objetos
* Métodos y funciones
* Manejo de estructuras de control
* Desarrollo de interfaces básicas
* Gestión de proyectos en Java

4. Ejecutar el archivo `Main.java`.
